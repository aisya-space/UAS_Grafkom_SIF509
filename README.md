## Panduan Kontrol Program
Program ini menggunakan Python PyOpenGL untuk mendemonstrasikan matriks transformasi.

### 1. Objek 2D (Persegi Hijau)
- **Panah Atas / Bawah**: Gerak Atas / Bawah (Translasi)
- **Panah Kiri**: Miring Samping (Shearing X)
- **Panah Kanan**: Miring Atas / Bawah (Shearing Y)
- **Tombol R / T**: Balik Objek (Refleksi / Cermin X & Y)

### 2. Objek 3D (Kubus Merah) & Umum
- **Tombol W / S**: Gerak Atas / Bawah
- **Tombol A / D**: Rotasi / Putar
- **SPACE**: Reset Semua Objek ke Posisi Awal
